# OOP Project
This is our group project during the course **TFB1033/TEB1043: Object Oriented Programming** where we chose to make an Inventory Management System.
The Objective of this project was to show OOP concepts in using <ins>Classes</ins>, <ins>Objects</ins>, <ins>Constructors</ins>, <ins>Inheritence</ins>, <ins>Polymorphism</ins> and ect.. The Web Application is coded in C#.

We also tested around with **Firebase Hosting** to test on **user authentication, data management and file storing**.
